## Welcome to my corner of the internet

For a while I wanted a single place to dump everything I care about — projects,
ideas mid-build, notes from things I'm reading, lessons from the on-call grind.
Here we are.

## Why this site

I work on distributed storage at **Marvell** and I love the messy intersection of
**reliability, performance, and good developer ergonomics**. The plan is to write
about that — plus the occasional side quest into machine learning and competitive
programming.

## The tech, briefly

- **Next.js + TypeScript** for the app shell
- **Tailwind v4** for styling, with custom CSS variables for the "Aurora Tape" palette
- **Framer Motion** for the gratuitous animations
- **MDX** for posts (this very thing)
- **Giscus** for the guestbook + comments, all backed by GitHub Discussions
- Hosted on **GitHub Pages** — push to `main` and it ships

The whole thing is statically exported, so there's no server to wake up. That's
intentional: portfolios should be instant.
